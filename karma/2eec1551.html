Web 和 Web 标准

Web（World Wide Web）即全球广域网，也称为万维网。

Web 标准：制作网页需要遵循的规范，是由 W3C 组织和其他标准化组织制定的一系列标准的集合。

  1. Web 标准包括 3 个方面：
    • 结构标准（HTML）：用于对网页元素进行整理和分类。
    • 表现标准（CSS）：用于设置网页元素的版式、颜色、大小等外观样式。
    • 行为标准（JS）：用于定义网页的交互和行为。
  2. Web 前端分 3 层：
    • HTML（HyperText Markup Language）：超文本标记语言，从语义的角度描述页面的结构。
    • CSS（Cascading Style Sheets）：层叠样式表，从审美的角度美化页面的样式。
    • JS（JavaScript）：从交互的角度描述页面的行为。

W3C 组织

W3C：World Wide Web Consortium，万维网联盟组织，用来制定 Web 标准的机构。

为什么要遵循 WEB 标准呢？因为浏览器的浏览器内核不同，会导致页面解析出来的效果出现差异，给开发者增加无谓的工作量。因此需要制作统一的标准。

浏览器是网页运行的平台，常见的浏览器有 Chrome、Safari、Firefox、IE、Edge、Opera 等。如下图所示：

组成

浏览器由 渲染引擎、JS引擎两部分组成。

  1. 渲染引擎（浏览器内核）

    渲染引擎（Rendering Engine）用来解析 HTML 和 CSS，它的作用就是读取网页内容，计算网页的显示方式并显示在页面上。

    浏览器 内核
    Chrome Blink
    Opera Blink
    360 极速浏览器 Blink
    Safari Webkit
    Firefox Gecko
    IE Trident

  2. JS 引擎

    JS 引擎用来解析网页中的 JavaScript 代码，浏览器本身并不会执行 JS 代码，而是通过内置 JS 引擎来执行 JavaScript 代码。JavaScript 语言属于脚本语言（逐行解释执行）。

    IE Trident
    Opera Linear A（4.0-6.1）/ Linear B（7.0-9.2）/ Futhark（9.5-10.2）/ Carakan（10.5-）

工作原理

  • User Interface：用户界面
  • Browser engine：浏览器引擎，用来查询和操作渲染引擎
  • Rendering engine：渲染引擎，负责解析 HTML、CSS
  • Networking：网络，负责发送网络请求
  • JavaScript Interpreter：JavaScript 解析器，负责解释执行 JavaScript 代码
  • UI Backend：UI 后端，用来绘制组合框和弹出框等组件
  • Data Persistence：数据持久化，数据存储 cookie、HTML5 中的 sessionStorage

初识HTML

HTML 全称为HyperText Markup Language，翻译为超文本标记语言，是一种描述性的标记语言，是负责描述文档语义的语言。

超文本

  • 图片、音频、视频、动画、多媒体等内容，称为超文本。因为它们已经超出了文本的限制。
  • 从一个文件跳转到另一个文件，与世界各地主机文件进行连接，即超级链接文本。

标记语言

  • 是一套标记标签。比如：标签<a>标识超链接、标签<img>标识图片、标签<h1>标识一级标题等等，它们都是属于 HTML 标签。
  • 标记语言是没有编译过程的，HTML 标签是直接由浏览器解析执行。

HTML 历史

XHTML（Estensible Hypertext Markup Language）：可扩展超文本标记语言。

XHTML 的主要目的是为了取代 HTML，可以理解为 HTML 的升级版。HTML 的标记书写很不规范，会造成其他设备无法正常显示。XHTML 与 HTML4.0 基本上一样。XHTML 是严格的、干净的HTML。

HTML 结构详解

HTML 标签通常是成对出现的（双边标记），比如<div>和</div>；也有少部分但标签，如：<br />、<hr />和<img src="" />等。

属性与标记之间、各属性之间需要空格隔开。

  1. 文档声明头

    标准的 HTML 页面，第一行以<!DOCTYPE ......>开头的语句。这一行就是文档声明头，即 DocType Declaration，简称 DTD。

    DTO 可告知浏览器文档使用哪种 HTML 或 XHTML 规范。

    HTML4.0.1 规定了HTML4.0.1和 XHTML 两大种规范：

    规范名称 规范细则
    HTML4.0.1 Strict：严格的(某些标签不能用，比如u)
    Transitional：普通的
    Frameset：带有框架的页面
    XHTML（小写标签、闭合、引号） Strict：严格的(某些标签不能用，比如u)
    Transitional：普通的
    Frameset：带有框架的页面

    Strict：这种模式里面的要求更严格。一些标签不能用，比如u标签（给文本增加下划线），这和HTML本质有冲突，因为HTML负责语义，不要负责样式。

    Transitional：表示“普通的”，这种模式就是没有一些别的规范。

    Frameset：表示”框架“，在框架的页面使用。

    XHTML1.0更为严格，规定标签必须小写、严格闭合标签、引号引起属性等等……

HTML标签

排版标签

  • 标题标签，<h1>定义最大标题，<h6>定义最小标签。
  • 段落标签<p>。
  • 水平线标签<hr \>。
  • 换行标签<br \>，文本强制换行显示。
  • <div>，可以把标签中内容分割为独立的区块，独占一行。
  • <span>，和div作用一致，但不换行。
  • 预定义标签<pre>，保留标签内部的所有空白字符。

文本级标签和容器级标签

  • 文本级标签：p、span、a、b、i、u和em。文本级标签里只能放文字、图片、表单元素。
  • 容器级标签：div、h系列、li、dt和dd。容器级标签可以放置任何元素。

<div>和<span>标签

div的语义是分割（division），span的语义是范围（span）。

div标签：可以把标签中的内容分割为独立的区块，独占一行。

span标签：和div的作用一致，但不换行。

PS：div属于容器级标签；span属于文本级标签。

字体标签

  • &nbsp：空格（non-breaking spacing，不断打空格）
  • &lt：小于号（less than）
  • &gt：大与号（greater than）
  • &amp：符号&
  • &quot：双引号
  • &apos：单引号
  • &copy：版权©️
  • &trade：商标™️
  • &#32464：文字绐（unicode编码）
  • <u>：下划线标记
  • <s>：中划线标记
  • <i>：斜体标记
  • <b>：加粗标记
  • <sup>和<sub>：上标标记和下标标记

超链接

  1. 外部链接：链接到外部文件

  2. 锚链接

    给链接起一个名字，作用是在本页面或其他页面的不同位置进行跳转。使用场景目录、回到顶部等。

  3. 邮件链接

属性值

  • href：目标URL
  • title：悬停文本
  • name：用于设置锚点名称
  • target：用什么方式打开浏览器
    • _self：在同一个网页中显示（默认值）
    • _blank：在新的窗口中打开
    • _parent：在父窗口中打开
    • _top：在顶级窗口中打开

图片标签

html页面不是直接插入图片，而是插入图片的引用地址，需要先把图片上传的服务器上。

图片属性

  • src可以是相对路径或绝对路径。
  • width：图像的宽度。
  • hight：图像的高度。
  • alt：当图片不可用时，代替（alternate）图片出现的内容。
  • title：提示性文本，鼠标悬停显示的文本。
  • align：图片和周围文字的相对位置。可以取值：bottom（默认）、center、top、left、right

列表标签

<ul>

  • ul：unordered list，无序列表。
  • li：list item，列表项。

属性值type可选disc（实心圆点，默认）、square（实心方点）、circle（空心圆点）

<ol>

  • ol：有序列表

属性值type可选1（阿拉伯数字，默认）、a、A、i、I。结合start="3"属性表示从3开始。

<dl>

定义列表，definition list，没有属性。dl的子元素只能是dt和dd。

  • <dt>：definition title，列表的标题。
  • <dd>：definition description，列表项。

表格标签<table>

表单标签<form>

用于与服务器交互，收集用户信息。

属性：

  • name：表单的名称。
  • id：表单的唯一标识符。
  • action：指定表单的处理程序。
  • method：表单的提交方式，一般取值get（默认）、post。

<input>文本框

属性：

  • type：文本类型，属性值可以是：
    • text（默认）
    • password：密码类型
    • radio：单选按钮
    • checkbox：多选按钮
    • checked：单选按钮或多选按钮处于选中状态
    • hidden：隐藏框，表单中包含不希望用户看到的信息
    • button：普通按钮
    • submit：提交按钮
    • reset：重置按钮，清空表单当前内容，并设置为最初的默认值
    • image：图片按钮
    • file：文件选择框。
  • value="xxx"：文本库的默认内容
  • size：文本库可以显示的字符数
  • readonly：文本只读框，不能编辑
  • disabled：文本框只读，不能编辑，光标点不进去

<marquee>滚动字幕标签

属性：

  • direction="right"：移动的目标方向。属性值可以是：left（从右向左移动，默认值）、right（从左向右移动）、up（从下向上移动）、down（从上向下移动）。
  • behavior="slide"：行为方式。属性值可以是：slide（只移动一次）、scroll（循环移动，默认值）、alternate（循环移动）、。 alternate和scroll属性值都是循环移动，区别在于：假设在direction="right"的情况下，behavior="scroll"表示从左到右、从左到右、从左到右···behavior="alternate"表示从左到右、从右到左、从左到右···
  • scrollamount="30"：移动的速度
  • loop="3": 循环多少圈。负值表示无限循环
  • scrolldelay="1000"：移动一次休息多长时间。单位是毫秒。

HTML规范

  1. html编写规范
    • HTML不区分大小写，但HTML标签名、类名、标签属性、大部分属性值建议统一小写。
    • HTML页面的后缀是html或htm。
  2. html基本语法
    • HTML对换行不敏感，对tab不敏感。
    • HTML文字之间有多个空格、换行、tab都将被折叠为一个空格显示。
    • 标签要严格封闭。
